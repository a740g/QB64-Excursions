Run 'Helicopter Rescue.exe' for menu screen                                     
                                                                                
Mouse:               Left button ................ Collective up                 
                     Right button ............... Collective down               
                     Mouse movement ............. Cyclic                        
                                                                                
Keyboard:            'A' ........................ Collective up                 
                     'Z' ........................ Collective down               
                     Arrow keys ................. Cyclic                        
                                                                                
Keys in menu:        'ESC' ... Exit to system                                   
                     '+' ..... Sound level up                                   
                     '-' ..... Sound level down                                 
                                                                                
Keys in game:        'SPC' ... Fire cannon                                      
                     'H' ..... Help                                             
                     'M' ..... Show map                                         
                     '.' ..... Next waypoint (0 = base)                         
                     ',' ..... Previous waypoint                                
                     'T' ..... View rescue tasks                                
                     'S' ..... View supplies available at base                  
                     'O' ..... View number of objects (sheds etc.) remaining    
                     'G' ..... Hide/show coordinates                            
                     'Q' ..... Hide/show cursor                                 
                     'B' ..... Hide/show rotor blades                           
                     'TAB' ... Toggle fullscreen                                
                     'C' ..... Toggle control type (mouse recommended)          
                     'I' ..... Invert control                                   
                     'P' ..... Pause                                            
                     'ESC' ... Abort game/replay                                
                                                                                
Keys in replay:      '-' ..... Slow down                                        
                     '+' ..... Speed up                                         
                                                                                
Macro recording is only available in mouse mode (mouse logger). Click in the    
text box in the menu screen to start the replay. For displays other than        
1920x1080 use 'ALT' + 'ENTER' to toggle through the different fullscreen modes  
until the result is as desired. Press 'TAB' during flight or in the menu to     
toggle windowed mode. When you don't end up in the Hall Of Fame, your flight    
recorder data will be lost. To uninstall the game delete the game-folder.

For lower specification machines try editing one or more of the following .TXT  
files in the folder '\TXTFiles':                                                
                                                                                
- SPEED.TXT     Change 'FASTCPU' to 'SLOWCPU'           Simplifies the models   
                                                        Doesn't effect the game 
                                                                                
- CURRENT.TXT   Change 'SCENE01.ITM' to 'SCENE00.ITM'   No model of the base    
                                                        Doesn't effect the game 
                                                                                
- MODE.TXT      Change 'NORMAL' to 'EASY'               No tilting calculations 
                                                        Doesn't effect the game 
                                                                                
- SKY.TXT	Change 'ON' to 'OFF'                    No blue sky, no stars   
                                                        Doesn't effect the game 
                                                                                
- RADAR.TXT     Change 'ON' to 'OFF'                    No radar calculations   
                                                        No damage counters      
                                                        Effects the game because
                                                        you won't have radar    
                                                        view or damage counters
